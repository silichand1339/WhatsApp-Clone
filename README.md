
# 💬 WhatsApp Web Clone

A modern **WhatsApp Web Clone** built using **HTML5** and **CSS3** as part of a **Full Stack Development Internship** team project.

The project focuses on recreating the **frontend user interface** of WhatsApp Web with a responsive design, clean layout, and professional styling without using JavaScript.

---

## 📸 Preview

> A static clone of WhatsApp Web featuring:
- Sidebar Navigation
- Chat List
- Chat Window
- Message Area
- Modern WhatsApp Dark Theme

---

## 🚀 Features

- ✅ WhatsApp Web inspired UI
- ✅ Fully responsive design
- ✅ Sidebar Navigation
- ✅ Chat List Section
- ✅ Conversation Window
- ✅ Beautiful message bubbles
- ✅ Search bar
- ✅ Filter buttons
- ✅ Footer message input
- ✅ Voice & Send buttons
- ✅ Smooth hover effects
- ✅ Modern dark theme
- ✅ HTML & CSS only (No JavaScript)
